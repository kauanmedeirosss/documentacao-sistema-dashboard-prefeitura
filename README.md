* O que temos em cada um dos 3 blocos da classe:
  - 1° -> Nome da classe (Em negrito)
  - 2° -> Atributos. Estrutura:
    * " - nomeAtributo : tipoAtributo "
  - 3° -> Métodos. Estrutura:
    * " + nomeMetodo(argumento : tipoArgumento) : tipoRetorno "
